# 打字速度检测网站

## 背景

开发一个面向用户的在线打字速度测试网站，帮助用户了解自己的打字速度（WPM）和准确率，并提供历史成绩记录与排行榜功能。

## 核心功能需求

### 1. 打字测试模块
- 系统随机展示一段文本（中文或英文，可切换）
- 用户在输入框中跟随输入，系统实时高亮正确/错误字符
- 计时从用户第一次击键开始，输入完成后自动结算
- 计算指标：WPM（每分钟字数）、准确率（正确字符数/总字符数）

### 2. 用户账号模块
- 支持邮箱+密码注册/登录
- 游客模式：无需登录可完成测试，但成绩不保存

### 3. 成绩记录模块
- 登录用户每次测试后自动保存成绩
- 个人历史成绩列表，支持按日期/WPM 排序
- 个人最佳成绩展示

### 4. 排行榜模块
- 全站排行榜（按最高 WPM 排序）
- 支持按语言（中文/英文）筛选
- 每页展示前 100 名

## 技术约束
- 前端：React + TypeScript
- 后端：Node.js + Express
- 数据库：PostgreSQL
- 实时性：打字过程中前端本地计算，无需实时请求后端
